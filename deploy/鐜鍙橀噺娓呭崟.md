# E23 后端环境变量清单

> 红线：**生产数据库密码、JWT 密钥、Token 加密密钥、各平台 Client Secret 一律只存在于服务器 .env，绝不写入 Git 仓库**（.env 已 gitignore）。

## 必填（缺省拒绝启动或生产降级保护）

| 变量 | 用途 | 生成建议 |
|---|---|---|
| `POSTGRES_PASSWORD` | 数据库密码 | `openssl rand -base64 24` |
| `JWT_SECRET` | JWT 签名密钥 | `openssl rand -base64 32` |
| `TOKEN_ENCRYPTION_KEY` | 第三方令牌 AES-256-GCM 加密密钥（生产缺省抛错） | `openssl rand -base64 32` |
| `CORS_ORIGIN` | 前端来源白名单（逗号分隔；生产禁止 `*`，缺省拒绝跨域） | 如 `https://e23.example.com,https://e23.pages.dev` |
| `FRONTEND_URL` | OAuth 回调完成后跳回的前端地址 | 如 `https://e23.example.com` |

## 数据库

| 变量 | 默认 | 说明 |
|---|---|---|
| `POSTGRES_USER` / `POSTGRES_DB` | e23 / e23 | compose 内网 |
| `DATABASE_URL` | 由 compose 组装 | 外部 PG 时显式给出 |
| `POSTGRES_PORT` | 5432 | 仅本机映射建议改 127.0.0.1 绑定 |

## 安全加固

| 变量 | 默认 | 说明 |
|---|---|---|
| `RATE_LIMIT_MAX` | 300 | 每 IP 每分钟请求上限，超限 429 |
| `RATE_LIMIT_WINDOW` | 1 minute | 限流窗口 |
| `CALLBACK_ALLOWED_HOSTS` | 空 | OAuth 回调主机白名单（逗号分隔），生产强烈建议配置 |

## 悦跑圈（首个真实联调平台，凭据到位后填）

| 变量 | 说明 |
|---|---|
| `JOYRUN_CLIENT_ID` / `JOYRUN_CLIENT_SECRET` | 悦跑圈开放平台应用凭据 |
| `JOYRUN_REDIRECT_URI` | `https://api.你的域名/api/providers/joyrun/callback`（需在悦跑圈登记白名单） |
| `JOYRUN_API_BASE` | 默认 `https://open.thejoyrun.com`，以官方文档为准 |
| `JOYRUN_SYNC_LOOKBACK_DAYS` | 首次同步回溯天数，默认 30 |

## 预留

| 变量 | 说明 |
|---|---|
| `MINIPROGRAM_API_KEY` | 微信小程序提交通道密钥（空=通道关闭，接口如实 503） |
| `STORAGE_ENDPOINT` / `STORAGE_BUCKET` / `STORAGE_ACCESS_KEY` / `STORAGE_SECRET_KEY` | 对象存储（凭证图片/异地备份），空=预留未启用 |
| `HUAWEI_*` / `GARMIN_*` | 后续平台（本阶段冻结，不新增适配器） |
| `BACKEND_PORT` | 后端端口，默认 8787 |

## 前端 / Android

| 变量 | 说明 |
|---|---|
| `VITE_API_BASE_URL` | 公网 API 地址，如 `https://api.你的域名`；为空=本机模式（不伪装已连接） |
