## 必购/必提供

| 资源 | 规格建议 | 参考费用 | 用途 |

## 部署动作（资源到位后）

```bash
# 1. 服务器装 Docker 后拉代码
git clone <仓库> && cd app/backend
# 2. 按 deploy/环境变量清单.md 填 backend/.env
# 3. 一键部署（构建+迁移+种子+健康检查）
bash ../deploy/deploy.sh
# 4. 配置 DNS 后启用 HTTPS
docker compose --profile https up -d
# 5. 备份 cron
crontab -e  →  0 3 * * * /path/deploy/pg_backup.sh
```
