## V1.0 (2026-07-08) — 路线冻结

### 路线确认
- **深圳闭环**：起点深圳 → 顺时针全国环线 → 终点深圳，48 个城市节点
- **总里程**：21,423 km（经 5 段实测数据校准）
- **跑步比例**：真实 1 km = 虚拟 10 km，完成全程需跑步约 2,142 km
- **路线骨架**：基于 G15/G1/G10/G6/G30/G219/G318/G214/G75/G72 等国家公路网真实国道/高速
- **新藏线保留**：叶城→阿里→日喀则（G219，极限难度）
- **方向**：顺时针（深圳 → 东部沿海 → 东北 → 内蒙 → 河西走廊 → 新疆 → 新藏 → 云南 → 贵州广西 → 海南 → 回归深圳）

### 数据结构变更

| 变更项 | 说明 |
|--------|------|
| 新增 `route_version: "1.0"` | 顶层版本标识，区别于 meta.version（数据格式版本） |
| 新增 `meta.status: "frozen"` | 标记路线已冻结，非经审批不得修改主路线节点 |
| 新增 `meta.frozen_at` | 冻结时间戳 |
| 新增 `sea_transfer` | 海上轮渡类型支持，描述 2 段琼州海峡轮渡通道 |
| 新增 `scenic_branch` | 景点分支数据结构预留（schema + 空数组），不增加主路线节点 |
| 更新 `route_type` | 北海/海口/三亚节点 route_type 从"高速公路+轮渡"统一为 `sea_transfer` |
| 更新 `statistics` | 新增 sea_transfer_count、longest_sea_segment、route_type_breakdown 调整 |
