## 二、数据结构确认

### 2.1 新增顶层字段

```json
{
  "route_version": "1.0",
  "meta": {
    "status": "frozen",
    "frozen_at": "2026-07-08",
    ...
  },
  "sea_transfer": [ ... ],
  "scenic_branch": {
    "description": "...",
    "branch_schema": { ... },
    "branches": []
  }
}
```

| 字段 | 状态 | 说明 |
|------|------|------|
| `route_version` | ✅ 已添加 | 顶层 1.0 标识，与 meta.version 区分 |
| `meta.status` | ✅ 已添加 | `"frozen"` 标记路线不可擅自修改 |
| `meta.frozen_at` | ✅ 已添加 | 冻结时间戳 |
| `sea_transfer` | ✅ 已添加 | 2 段琼州海峡轮渡，含港口/里程/描述 |
| `scenic_branch` | ✅ 已添加 | schema 预留 + 空 branches 数组 |
| `route_type` 更新 | ✅ 已完成 | 北海/海口/三亚 → `sea_transfer` 类型 |

### 2.2 sea_transfer 明细

| ID | 名称 | 路线 | 海上公里 | 含公路总距 |
|----|------|------|----------|-----------|
| `sea_beihai_haikou` | 北部湾轮渡 | 北海港 → 海口港/新海港 | ~30 km | 250 km |
| `sea_hainan_return` | 琼州海峡轮渡 | 三亚→G98→海口→新海港→徐闻港→湛江 | ~30 km | 460 km |

> 注：`sea_hainan_return` 中徐闻为湛江下辖县，不增加主路线城市节点（48 城不变），在湛江节点中集中表示。

### 2.3 scenic_branch 预留

```json
{
  "branch_id": "string",
  "parent_node_id": "string",
  "name": "string",
  "scenic_spots": ["string"],
  "detour_km": "number",
  "route_type": "string",
  "difficulty": "string",
  "description": "string"
}
```

- 当前 `branches: []` 为空，后续可按需扩展景点支线
- 不影响 48 主节点，不改动路线总里程
