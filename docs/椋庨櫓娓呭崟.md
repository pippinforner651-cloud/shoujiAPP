# E23 风险清单（截至 2026-07-20）

## 一、凭据与商务风险
| 风险 | 等级 | 说明与对策 |
|---|---|---|
| 悦跑圈凭据申请被拒或周期不可控 | 高 | 首平台联调全部依赖官方审批；过渡方案：GPX/凭证导入已可用 |
| 悦跑圈接口字段/分页/刷新规则与公开资料不符 | 中 | 差异收敛在 normalizeActivity 单点函数，联调时按正式文档修正 |

| 华为 Health Kit 审核严格 | 中 | 已冻结，企业认证周期较长 |

## 二、上线运维风险
| 风险 | 等级 | 说明与对策 |
|---|---|---|
| 云资源未采购，后端未上线 | 高 | 资源清单与一键部署脚本已就绪，采购后约半小时上线 |
| 密钥管理不当 | 中 | compose 强制必填、.env 不入仓、TOKEN_ENCRYPTION_KEY 生产缺省拒启动 |
| 单机部署单点故障 | 中 | pg_backup.sh 每日备份+保留14份；对象存储异地容灾已预留 |
| 限流误伤 | 低 | 默认 300/分/IP，可按班级规模调 RATE_LIMIT_MAX |
