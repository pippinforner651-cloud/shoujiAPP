# 可迁移功能清单（迁入 GitHub 后即可用 / 需后端 / 需原生）

## A. 迁入即可用（纯前端，本机数据）

1. 环中国真实路线地图：27,171 km、130 命名站点 + 46 边界途经点、两指缩放/拖动
2. 跑量 1:1 点亮路线（已完成橙色 / 未跑蓝色 / 剩余里程）
3. 当前位置：所在道路、正在前往、距下一城市、跑动小人（可点击）
4. 站点详情：真实道路、环线里程、名胜古迹、美食推荐
5. 跑步：前台 GPS（漂移过滤、拒权明示）、演示模式、手动补录、暂停/继续/结束、跑后总结、本地历史（关闭应用保留）
6. 首页十要素：今日里程、今日打卡次数、累计、当前道路、下一城市、地图三态、最近真实动态、年度目标 270km 进度
7. 我的：头像/昵称（可改）、累计、运动次数、平均配速、环线贡献比例、历史记录
8. 地图包系统：导入第三方包、完整性校验值、一键回滚、全球跑/高德接口预留
9. PWA：添加到主屏幕（iOS/Android）

## B. 需多人后端（接口已预留，前端有闸门 MULTIPLAYER_ENABLED）

1. 微信授权登录（code2session，需开放平台资质）
2. E23 邀请码 + 管理员审批
3. 真实排行榜（头像/昵称/年度/月度/打卡/配速）
4. 全班实时累计、今日参与人数
5. 跑步记录云同步与运动审核

4. 佳明/华为/苹果手表（Garmin Health API / Health Connect / HealthKit）
5. iOS 原生 / TestFlight
