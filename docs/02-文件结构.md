## 数据流

```
用户输入里程
    │
    ▼
DistanceInput ──→ useRunStore.addRun()
    │
    ├──→ localStorage 持久化
    │
    ├──→ useRouteProgress 计算当前位置
    │       │
    │       ├──→ ChinaMap 更新跑者位置
    │       └──→ 检测是否到达新城市
    │               │
    │               └──→ CityCard 弹窗（启用则解锁）
    │
    └──→ useAchievement 检测成就解锁
            │
            └──→ BadgeCard 弹窗（启用则解锁）
```

## 关键文件说明

| 文件 | 说明 |
|------|------|
| `src/data/route.ts` | **核心数据文件**，定义全部 48 城市 + 9 里程碑 |
| `src/utils/storage.ts` | 数据持久化封装，版本号管理，自动迁移 |
| `src/stores/useRunStore.ts` | 跑步数据状态中枢，所有页面通过它读写 |
| `src/components/map/ChinaMap.tsx` | 地图核心，初始化一次，Props 驱动更新 |
| `src/types/index.ts` | 全局类型，所有模块的类型来源 |
