## 数据兼容与 P0 风险

包名已从 `com.chinarun.app` 改为 `com.e23running.app`（修改发生在 `fb19a66` 与 `2f5f02b`）。Android 会将它们视为不同应用；旧 APK 的 Capacitor WebView 私有 localStorage 不会被普通前端 JavaScript 自动读取或迁移。除非用户确认旧 APK 只用于少量测试，否则这应作为 P0 风险处理。

## Android 构建基线

- Java：Microsoft OpenJDK 17.0.19。
- Gradle Wrapper：8.14.3；首次诊断时下载不完整，后已下载并解压。
- 离线构建在 6 秒内明确失败：本机缓存缺少 `com.android.tools.build:gradle:8.13.0` 和 `com.google.gms:google-services:4.4.4`。
- Gradle、Google Android SDK 与 Maven Central 的公开地址均可达；在线初次依赖解析在诊断时限内未进入编译任务。
- `frontend/android/local.properties` 不存在；常见 Android SDK 目录不存在，`adb` 不在 PATH。
- 因此尚不能称本机 APK 构建已通过；后续需安装或指定 Android SDK，再执行带 `--console=plain --info --stacktrace` 的构建。
