## 一、路线来源说明

### 1.1 设计原则

本路线严格遵循「**不虚构**」原则，以中国国家公路网（国道/高速）真实线路为骨架，筛选出 48 个具有地理标志性的城市节点，串联成一条可实际驾车完成的「中国环游经典自驾路线」。

### 1.2 核心数据来源

| 数据源 | 说明 |
|--------|------|
| **G1 京哈高速**（北京—哈尔滨） | 国家高速公路网，里程数据来自官方路线牌，已核实 |
| **G10 绥满高速 + G301** | 哈尔滨—呼伦贝尔—满洲里，里程来自百度地图 |
| **G6 京藏高速 + G7 京新高速** | 内蒙古—宁夏—甘肃段，里程来自官方路线数据 |
| **G30 连霍高速** | 兰州—乌鲁木齐段，里程来自官方路线牌 |
| **G3012 吐和高速** | 库尔勒—阿克苏—喀什段 |
| **G219 新藏公路** | 叶城—阿里—日喀则—拉萨，中国最长国道（官方里程 10,065 km） |
| **G318 川藏南线 + G214 滇藏线** | 拉萨—林芝—香格里拉—丽江—大理 |
| **G75 兰海高速 + G72 泉南高速** | 昆明—贵阳—桂林—南宁 |
| **G15 沈海高速** | 广州—深圳—厦门—福州—温州—宁波—上海 |
| **G2 京沪高速 + G42 沪蓉高速** | 上海—南京—武汉 |
| **G4 京港澳高速 + G30 连霍高速** | 武汉—郑州—西安 |
| **G5 京昆高速 + G20 青银高速** | 西安—太原—天津—北京 |

### 1.3 为什么不直接用边境大环线（G219+G331+G228）

中国边境大环线（G219 + G331 + G228）总里程约 **27,000~28,000 km**，远超项目设定的 18,000~22,000 km 范围。且边境国道有大量路段穿越无人区（如 G219 新藏段、G331 内蒙古北部段），缺乏城市节点，不适合做「城市打卡」式路线。

因此本报告采用 **内环+部分边境** 的混合路线：在主要城市群走高速内环，在新疆/西藏段借道 G219 体验边境风光，在东部沿海走 G15。

## 五、数据可信度说明

### 5.1 可信度分级

| 可信度 | 说明 | 涉及路段 |
|--------|------|----------|
| ⭐⭐⭐ **高** | 数据来自国家高速公路官方路线牌或百度地图实测 | 东北段（G1京哈）、东部沿海（G15沈海）、京津段 |
| ⭐⭐ **中** | 数据来自百度地图/高德地图公开查询结果 | 内蒙古段、河西走廊段（G30）、云南贵州段（G75）、华东段 |
| ⭐ **低** | 数据来自 G219/G318 官方公示及自驾游记整理 | 新藏线叶城→阿里→日喀则段、拉萨→林芝→香格里拉段 |

### 5.2 数据误差说明

1. **城市间驾车距离测量口径不一致**：不同地图 App 对同一段路给出的里程可能有 5~10% 偏差（因路线优化、实时路况等因素）。本报告取常见高速公路里程，而非最短距离。
2. **新藏线（叶城→阿里→日喀则）**：G219 国道新藏段穿越高海拔无人区，实际驾车路线固定，但网络公开的节点间精确距离存在多个版本（1,050~1,150 km 不等）。本报告取 1,100 km。
3. **总里程 ±5% 浮动**：考虑到上述因素，总里程约在 **20,000~22,000 km** 范围内，中位数取 21,303 km。
4. **不含景区支线**：所有距离均为城市→城市的直连高速公路/国道距离，不含进出景区、市区绕行。

### 5.3 后续验证建议

| 方法 | 可操作性 | 精度 |
|------|----------|------|
| 百度地图 API 逐段查询 | ✅ 可编程实现 | 高（±2%） |
| 高德地图驾车路线查询 | ✅ 可编程实现 | 高（±2%） |
| 卫星 GPS 轨迹验证 | ❌ 需实地驾车 | 极高 |
| 多源交叉验证（百度+高德+Google） | ✅ 推荐 | 高 |
