## 10. 重复数据测试

| 检测策略 | 实现 | 效果 |
|----------|------|------|
| sourceActivityId 精确匹配 | ✅ 最高优先级，confidence=1 | 同一平台活动 ID 不会重复 |
| rawDataHash 匹配 | ✅ 第二优先级，confidence=0.98 | 相同数据内容不会重复 |
| 时间+距离模糊匹配（±5%） | ✅ 第三优先级，confidence=0.9 | 手动录入同一次跑步不会重复 |
| 重复标记 | ✅ verificationStatus = 'duplicate' | 重复数据不纳入排行榜统计 |

**验证方法：**
- 同一 GPX 文件导入两次 → 第二次被 rawDataHash 拦截 ✅
- 同一手动记录提交两次 → 被时间+距离模糊匹配拦截 ✅
