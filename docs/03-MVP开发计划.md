# 全民环游中国虚拟跑步地图 — MVP 开发计划 V2

> 编写日期：2026-07-08 | 状态：已确认
> 修订依据：总控审核 → 确定 React+Vite、48 城路线、1:10 比例、MVP 功能清单

---

## 一、MVP 目标

**一句话**：用 React + Vite 搭建一个 Web 应用，用户输入跑步里程后，在 ECharts 中国地图上沿 48 城自驾路线推进，城市打卡 + 成就解锁。

**必须交付（7 项）**：
1. ✅ 用户昵称设置
2. ✅ 本地 localStorage 数据持久化
3. ✅ 输入每日跑量
4. ✅ 自动计算累计跑量
5. ✅ 地图路线推进（ECharts 中国地图 + 路线连线 + 跑者标记）
6. ✅ 当前到达城市显示（城市打卡卡片）
7. ✅ 个人统计页面

**暂不开发**：
- ❌ GPS 自动同步
- ❌ 微信登录 / OAuth
- ❌ 支付 / 商业化
- ❌ 后端 API
- ❌ 好友 / 排行榜
