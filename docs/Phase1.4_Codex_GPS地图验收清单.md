# E23跑起来 Phase 1.4 GPS地图验收清单

本清单用于 Android 预览版真机验收。自动测试与 APK 构建通过不等于真机 GPS、锁屏和后台能力已经通过。

## 安装与启动

- 安装后连续冷启动 5 次，不闪退。
- 前后台切换 3 次，页面和本地记录保持。
- 拒绝精确位置权限时不创建跑步活动，并显示真实原因。
- 关闭 GPS 系统开关时不创建跑步活动，并提示打开定位。

## 户外跑

- 点击开始后进入“正在获取真实 GPS”，SQLite 尚无活动记录。
- 首个合格 GPS 点只建立基线，不累计距离。
- 确认开始只生成一个 `clientActivityId`；重复点击不生成第二条活动。
- 实走至少 200 米，地图当前位置、轨迹、距离、时长和配速持续更新。
- NETWORK/PASSIVE、模拟定位、低精度、时间倒退和异常速度点不累计距离，但保留审计原因。
- 暂停、继续、结束保存可用；结束后从 SQLite 重读完整轨迹。

## 锁屏与异常恢复

- 跑步中锁屏 5 分钟后解锁，检查 GPS 回调、SQLite 写入和距离连续性。
- 强制划掉应用后重新打开，显示未结束活动并提供继续、结束保存、放弃留痕。
- 放弃不删除 SQLite 轨迹，记录状态为 `ABANDONED`。

## 地图与室内跑

- 地图瓦片失败时显示“地图暂时无法加载，GPS仍在记录”，跑步不中止。
- 拖动地图关闭跟随，“重新定位”恢复跟随。
- 室内跑不请求 GPS，结束时只能按跑步机真实距离保存。
- 热量明确为估算值；目标达成不自动结束，清除目标后归零。

## 安全边界

- 不启用真实微信、生产数据库、正式多人或正式路线。
- 不执行数据库迁移，不提交 `.env`、Token、签名私钥或本机配置。
- V2 路线保持 `DRAFT`，不生成假跑者、假排行或假全班进度。
